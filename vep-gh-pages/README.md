# Visual Effects + (GitHub Pages Site)

This repo hosts the public site for the resource pack.

## Local preview
1) Install Ruby and Bundler.
2) `bundle init` then add `gem "github-pages", group: :jekyll_plugins` to Gemfile.
3) `bundle install`
4) `bundle exec jekyll serve`

Or enable **GitHub Pages**: Settings → Pages → Build from `main` (root).
