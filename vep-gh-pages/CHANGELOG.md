# Changelog / История изменений
Все заметные изменения ресурспака будут документироваться в этом файле.

## [0.4.0] — 2025-08-16
### Added
- Meteor shower particle system (fast, high‑alt with trails).
- Aurora rings v2 with phase time distortion.
- Water splash & ripples refresh.
- New colormaps for dusk/sunset.

### Fixed
- Correct `DX/DY/DZ` usage, `TIME` instead of `GAMETIME`.
- `rand()` / `gaussian()` arguments removed (scale instead).
