# Visual Effects +

Visual Effects + is a Minecraft resource pack focused on improving visual effects, particles, atmosphere, lighting, weather, and environmental details.

## Features

- Enterily new particles with their own spawn events
- Improved vanilla particles
- Custom colormaps
- Enhanced fog and sky colors
- Sky shaders
- Better weather visuals
- Polytone post shaders
- Polytone config screen

## Requirements

- Minecraft Java Edition
- [Polytone](https://modrinth.com/mod/polytone)

## Installation

1. Download the latest .zip from Releases.
2. Put it into .minecraft/resourcepacks.
3. Enable it in Minecraft resource pack settings.
4. Install latest version of [Polytone](https://modrinth.com/mod/polytone)

## Compatibility

This resource pack does not change gameplay.  
Some shaders can broke existing particles.
